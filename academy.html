Foundations

What is Runlayer

One platform, one golden path to enable and control every MCP, Skill, and Agent.

Product 101

Runlayer is an AI infrastructure platform with security built in, not a bolt-on tool.

  • MCP & connectors: governed access to tools, enforced by policies and deterministic guardrails.
  • Agent Sessions: real-time visibility + audit of every agent action; intent-drift detection.
  • Security scanning: ML models on GPUs catch prompt injection and leaked tokens in real time.
  • Agent Accounts: scoped non-human identities with delegation (RFC 8693 token exchange).
  • Runlayer Deploy: host custom MCP servers via CLI + runlayer YAML → ECS Fargate / K8s.
  • Skills & Plugins: distributable capabilities (plugins are being unified into connectors).

How we work

Builder culture: direct, no fluff, over-communicate. Ship early and iterate. Customer obsession (~5-min Slack response). Spec-driven, pod-based.

The golden path

We don't sell a platform. We take customers from where they are to AI-native as fast as possible. The same path runs your onboarding.
